# PhonePay_analysis_dashboard
A Power BI-based PhonePe Transaction Data Analysis Dashboard that analyzes digital payment transactions to provide meaningful insights into transaction trends and user activity. 
The dashboard transforms raw transaction data into interactive visualizations, enabling analysis of transaction volume, transaction value, geographical distribution, and payment patterns across different categories and time periods. 
It helps identify key trends, compare performance across regions, and support data-driven decision-making through dynamic reports. 
This project demonstrates the use of Power BI, Power Query (ETL), DAX, Data Modeling, Data Cleaning, Interactive Dashboards, Maps, Charts, KPIs, Filters/Slicers, and Data Visualization to convert raw transaction data into actionable business insights
